== Description ==

Use the power of custom styled text widgets in your designs. Display your widgets in fancy style by using many options in widget settings.

Each widget can be customized differently.

Some of the widget settings:

*  Use all google fonts just by selecting from dropdown
*  Preview google fonts in widget area
*  Change widget content padding for easier fit to your design!
*  Change widget background with predefined backgrounds or even upload your own
*  Add awesome CTA button to your widget to generate easier user interaction
*  Modify your CTA button to fit your overall site design
*  Change font colors on the fly with color picker
*  Use our predefined pins for cool & slick look
*  Position pins however you need them


== Installation ==

This section describes how to install the plugin and get it working.

1. Install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Setup your widget in widgets section in WordPress Dashboard.
4. Enjoy!
